# Kai Maison Static Clone

Static Vite + React clone of Kai Maison Berlin using local optimized assets.

Live preview is published through GitHub Pages:

https://hoanghaip2005.github.io/kaimaison-clone/

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Cloud Run

```bash
gcloud run deploy kaimaison-clone --source . --region asia-southeast1 --allow-unauthenticated
```
